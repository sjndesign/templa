# Templa

## 디자인을 빠르게, 그리고 일관되게 만드는 서비스

Templa는 디자인 경험이 없어도 브랜드를 빠르게 만들 수 있는 디자인 자동화 서비스입니다.

---

## 🚀 서비스 소개

디자인 시스템을 입력하면  
명함, 이메일 서명, SNS 프로필 이미지가 자동으로 생성됩니다.

모든 결과물은 동일한 디자인 규칙을 기반으로 만들어져  
브랜드의 일관성을 유지할 수 있습니다.

---

## 🎯 해결하는 문제

- 디자인을 매번 새로 만들어야 하는 문제
- 브랜드 디자인이 일관되지 않는 문제
- 디자이너 없이 브랜드를 운영하기 어려운 문제

---

## ⚙️ 핵심 기능

- 디자인 시스템 입력
- 브랜드 정보 설정
- 템플릿 자동 생성
- 실시간 미리보기
- PNG / PDF 다운로드

---

## 👥 대상 사용자

- 1인 사업자
- 스타트업 팀
- 마케팅 담당자
- 디자이너가 없는 기업
- 개발자

---

## 💡 핵심 가치

- 빠른 디자인 생성
- 브랜드 일관성 유지
- 누구나 사용할 수 있는 디자인 시스템
