# F-006 브랜드 수정

## 1. 기능 개요

사용자가 생성한 브랜드의 정보를 수정한다.

---

## 2. 목적

- 사용자가 생성한 브랜드를 수정한다.

---

## 3. 관련 화면

| 화면 ID | 화면명 |
|----------|---------|
| SC-006 | 브랜드 목록 |
| SC-008 | 브랜드 상세 |

---

## 4. 사용자

- 로그인한 사용자

---

## 5. 입력 데이터

| 항목 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 검색어 | String | X | 브랜드명을 검색한다. |
| 업종 | String | X | 업종별 브랜드를 조회한다. |
| 정렬 | String | X | 최신순, 이름순 등 |

---

## 6. 처리 과정

1. 브랜드 목록 화면에 진입한다.
2. 시스템이 사용자의 브랜드 목록을 조회한다.
3. 사용자가 검색어를 입력한다.
4. 사용자가 업종을 선택한다.
5. 시스템이 조건에 맞는 브랜드를 조회한다.
6. 조회 결과를 화면에 표시한다.

---

## 7. 출력 결과

### 성공

- 조회된 데이터를 표시한다.

### 실패

- 검색 결과가 없습니다.는 메시지를 표시한다.

---

## 8. 예외 처리

| 상황 | 처리 |
|------|------|
| 검색 결과 없음 | 검색 결과가 없습니다. |
| 서버 오류 | 잠시 후 다시 시도해주세요. |

---

## 9. 권한
