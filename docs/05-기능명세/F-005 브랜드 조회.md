# F-005 브랜드 조회

## 1. 기능 개요

사용자가 생성한 브랜드 정보를 수정한다.

---

## 2. 목적

- 브랜드 정보를 최신 상태로 유지한다.
- 브랜드 정보를 수정할 수 있도록 한다.
- 수정된 정보를 디자인 시스템과 템플릿에 반영한다.

---

## 3. 관련 화면

| 화면 ID | 화면명 |
|----------|---------|
| SC-006 | 브랜드 목록 |
| SC-007 | 브랜드 생성 |
| SC-008 | 브랜드 상세 |

---

## 4. 사용자

- 로그인한 사용자

---

## 5. 입력 데이터

| 항목 | 타입 | 필수 | 설명 |
|------|------|------|------|
| brandId | Number | O | 수정할 브랜드 ID |
| 브랜드명 | String | O | 브랜드명 |
| 브랜드명 영문 | String | O | 브랜드 영문명 |
| 업종 | String | X | 업종 |
| 브랜드 설명 | String | X | 브랜드 설명 |

---

## 6. 처리 과정

1. 브랜드 상세 화면에 진입한다.
2. 수정하기 버튼을 클릭한다.
3. 브랜드 정보를 수정한다.
4. 저장 버튼을 클릭한다.
5. 시스템이 입력값을 검증한다.
6. 시스템이 브랜드 정보를 저장한다.
7. 수정 완료 메시지를 표시한다.
8. 수정된 정보를 화면에 표시한다.

---

## 7. 출력 결과

### 성공

- 브랜드 정보가 수정되었습니다.
- 수정된 브랜드 정보를 표시한다.

### 실패

- 브랜드 정보 수정에 실패했습니다.

---
