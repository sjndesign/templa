# F-004 브랜드 생성

## 1. 기능 개요

사용자가 새로운 브랜드를 생성하여 디자인 시스템과 템플릿을 관리할 수 있도록 한다.

---

## 2. 목적

- 새로운 브랜드를 생성한다.
- 브랜드별 디자인 시스템을 관리할 수 있도록 한다.
- 생성된 브랜드를 기반으로 템플릿을 제작할 수 있도록 한다.

---

## 3. 관련 화면

| 화면 ID | 화면명 |
|----------|---------|
| SC-007 | 브랜드 생성 |

| SC-008 | 브랜드 상세 |

---

## 4. 사용자

- 로그인한 사용자

---

## 5. 입력 데이터

| 항목 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 브랜드명 | String | O | 브랜드명 한글 |
| 브랜드명 영문 | String | O | 브랜드명 영문 |
| 업종 | String | X | 브랜드의 업종 |
| 브랜드 설명 | String | X | 브랜드에 대한 설명 |

---

## 6. 처리 과정

1. 브랜드명을 입력한다.
2. 브랜드 영문명을 입력한다.
3. 업종을 선택한다.
4. 생성하기 버튼을 클릭한다.
5. 시스템이 입력값을 검증한다.
6. 시스템이 브랜드명 중복 여부를 확인한다.
7. 중복될 경우 브랜드명 뒤에 "(1)", "(2)" 등을 자동으로 추가한다.
8. 브랜드 정보를 저장한다.
9. 디자인 시스템 설정 화면으로 이동한다.

---

## 7. 출력 결과

### 성공

- 브랜드 생성 완료 메시지를 표시한다.
- 디자인 시스템 설정 화면으로 이동한다.

### 실패

- 오류 메시지를 표시한다.
- 브랜드 생성 화면을 유지한다.
