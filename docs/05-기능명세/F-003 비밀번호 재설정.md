# F-003 비밀번호 재설정

## 1. 기능 개요

사용자가 본인 인증 후 새로운 비밀번호로 변경할 수 있도록 한다.

---

## 2. 목적

- 본인 인증을 통해 비밀번호를 변경한다.
- 계정 보안을 유지한다.
- 변경된 비밀번호로 다시 로그인할 수 있도록 한다.

---

## 3. 관련 화면

| 화면 ID | 화면명 |
|----------|---------|
| SC-004 | 비밀번호 찾기 |
| SC-002 | 로그인 |

---

## 4. 사용자

- 비회원

---

## 5. 입력 데이터

| 항목 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 이름 | String | O | 사용자 이름 |
| 이메일 | String | O | 가입한 이메일 |
| 새로운 비밀번호 | String | O | 계정 비밀번호 |
| 비밀번호 확인 | String | O | 새로운 비밀번호 확인 |

---

## 6. 처리 과정

1. 이름을 입력한다.
2. 이메일을 입력한다.
3. 새로운 비밀번호를 입력한다.
4. 비밀번호 확인을 입력한다.
5. 시스템이 입력값을 검증한다.
6. 가입된 사용자 정보를 확인한다.
7. 비밀번호를 변경한다.
8. 변경 완료 메시지를 표시한다.
9. 로그인 화면으로 이동한다.

---

## 7. 출력 결과

### 성공

- 비밀번호 변경 성공 메시지
- 로그인(SC-002) 이동

### 실패

- 오류 메시지를 표시한다.
- 비밀번호 재설정 화면을 유지한다.

---

## 8. 예외 처리

| 상황 | 처리 |
|------|------|
| 이메일 미입력 | "이메일을 입력해주세요." |
| 비밀번호 미입력 | "비밀번호를 입력해주세요." |
| 이메일 형식 오류 | 이메일 형식을 확인해주세요. |
| 존재하지 않는 계정 | 가입된 계정이 없습니다. |
| 비밀번호 불일치 | 새 비밀번호와 비밀번호 확인이 일치하지 않습니다. |
| 서버 오류 | 잠시 후 다시 시도해주세요. |

---

## 9. 권한

- 비회원만 접근 가능
- 로그인된 사용자는 비밀번호 재설정 페이지 접근 시 대시보드로 이동한다.
